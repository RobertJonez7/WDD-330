General Notes



-The ‘this’ keyword in JavaScript can change depending on its scope. There are six different values that ‘this’ can take on. They are:

1. this in global context
2. this in object construction
3. this in an object method
4. this in a simple function
5. this in an arrow function
6. this in an event listener


-Global context: refers to window object
-Object construction context: When you use the ‘new’ keyword, ‘this’ refers to the instance.
-Object Method context: refers to the object itself
-Simple function context: refers to window object
-Arrow function context: refers to the object itself
-Event listener context: refers to the element that fired the event


-DOM (Document Object Model)
-It defines what you see on the screen (The HTML elements, colors, etc.)
-JavaScript can manipulate the DOM, that is how you are able to insert/remove things on the screen using JavaScript.


-Events are when things happen to HTML elements.
-One example of an event is when you click on something on a webpage. Or if you scroll down. Or if you minimize the screen. There is many events that you can utilize.
-Typically, you want something to happen when an event is fired. You can do this using addEventListener in JavaScript. Or you can add it directly to an HTML element (onclick, onchange, etc.)
