Forms


The form element is used to extract information from user inputs such as text boxes, checkboxes, radio buttons, etc. When a user inputs information that we want to use later, we can pass it through the form element and get that information to use it later. We can use JavScript for this. Here are some examples of elements we can use inside of a form:
  • input type="text"
  • input type="radio"
  • input type="checkbox"
  • input type="submit"
  • input type="button"
Some applications for this would be in an ecommerce site. If I checked all of the stuff I wanted to buy, then I could use a form to send the checked boxes to the backend where we can process what was clicked.
